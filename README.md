# EAW_Substance_Painter_Resources
 Templates and shaders for easier Substance Painter to EAW workflow


# How to install temaples and shaders
Copy /assets/ files into susbtance user/documents/adobe/Susbtance Painer/assets directory

or

Download /assets/ into storage folder then add the newly created folder to resource directory in Substance Painter settings


# Changing Shader Parameters
The shader has some parameters, these can be found in the Shader Settings tab.
